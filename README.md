# Abyss Branch

Abyss Branch は、Godot 4.6 で開発するカードベースのローグライクゲームプロジェクトです。対応プラットフォームは Android と iOS です。

このリポジトリには、企画からリリースまでの開発に必要なソースコード、設計ドキュメント、開発ツールが含まれています。

## 目的

- Godot 4.6 を用いて、完成されたカードベースのローグライクゲームを開発する。
- 可能な限り実装前に、ゲームデザイン、ルール、データを定義する。
- 設計、シミュレーション、実装を密接に同期させる。
- 実験的なアーキテクチャよりも、プレイ可能なゲームを完成させることを優先する。

## リポジトリ構造

```text
abyss-branch/
├─ README.md
├─ godot/
├─ data/
├─ sim/
├─ tools/
└─ docs/
```

## ディレクトリ説明

- `godot/`  
  ゲーム本体。実行ロジック、シーン、アセットを管理する中核ディレクトリ。

- `data/`  
  カード・敵・報酬などのゲームデータ定義。バランス調整の基準となる元データ。

- `sim/`  
  シミュレーション実行と結果分析のための領域。Godot ヘッドレス実行の制御もここで扱う。

- `tools/`  
  開発補助ツール置き場。データ検証、変換、生成、QA支援などの非本体処理。

- `docs/`  
  仕様・方針・運用ルールのドキュメントを集約するディレクトリ。

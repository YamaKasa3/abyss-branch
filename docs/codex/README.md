# コーデックス文書

このディレクトリは、Codex の運用文書を管理する場所です。

## 含める文書
- `rules.md`: 文書運用ルールの正本
- `review-checklist.md`: 文書レビュー時の確認項目
- `templates/spec-template.md`: 仕様文書テンプレート
- `templates/procedure-template.md`: 手順文書テンプレート
- `templates/policy-template.md`: 方針文書テンプレート
- `../CHANGELOG.md`: リリース単位の変更履歴（自分向け）

## 運用方針
- 変更時は関連文書を同時に更新する
- 実運用の判断基準は `rules.md` を正とする
